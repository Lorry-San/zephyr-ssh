---

Bug 列表（按优先级排序）


---

Bug #01：数据未持久化保存（P0 - 阻断级）

问题描述：当前系统未对数据进行持久化存储，页面刷新或更新后数据即丢失

影响范围：所有涉及数据保存的功能

期望结果：数据应在刷新或更新后仍能正常保留



---

Bug #02：跳板机功能无效（P0 - 阻断级）

问题描述：

不支持多跳（多级跳板）

跳板机及代理配置未实际生效


影响范围：SSH 跳板机及代理相关功能

期望结果：

支持多跳配置

配置后应实际影响连接路径并生效




---

Bug #03：F2A 功能无法使用（P1 - 高优先级）

问题描述：双因素认证（F2A）功能无法正常启用或验证失败

影响范围：账号安全相关功能

期望结果：F2A 应可正常配置并完成验证流程



---

Bug #04：Passkey 功能不可用（P1 - 高优先级）

问题描述：Passkey 未调用浏览器 WebAuthn 接口，导致功能无法使用

影响范围：无密码登录相关功能

期望结果：正确调用浏览器接口并完成认证流程



---

Bug #05：移动端 SSH 界面显示异常（P1 - 高优先级）

问题描述：移动端唤起键盘后，SSH 界面与键盘之间出现黑边，且过渡动画不流畅

影响范围：移动端 SSH 使用场景

期望结果：界面无黑边，键盘弹出过渡自然流畅



---

Bug #06：深色模式未正确联动及系统主题适配缺失（P1 - 高优先级）

问题描述：

主界面切换深色模式时，SSH 页面未同步切换

SSH 页面切换主题不会反向影响主界面（该行为可保留）

未支持根据系统主题（深色 / 浅色）自动切换


影响范围：主题系统 / 全局 UI 一致性

期望结果：

主界面切换深色时同步 SSH 页面

支持跟随系统主题自动切换（prefers-color-scheme）




---

Bug #07：UI 动画缺失（P2 - 中优先级）

问题描述：

模式切换按钮缺乏动画效果

标签页关闭无过渡动画


影响范围：整体 UI 交互体验

期望结果：关键交互应具备完整过渡动画



---

Bug #08：全屏动画体验不佳（P2 - 中优先级）

问题描述：全屏切换动画速度过快，视觉效果不自然

影响范围：全屏切换功能

期望结果：动画节奏合理，视觉效果流畅（参考系统级动效）



---

Bug #09：全屏策略设计不合理（P2 - 中优先级）

问题描述：当前全屏逻辑未区分设备类型

影响范围：全屏功能

期望结果：

移动端：使用自定义全屏

桌面端：使用浏览器原生全屏




---

Bug #10：终端标签页占用空间过大（P2 - 中优先级）

问题描述：

存在“多标签页工作区”等冗余文字

标签页占用空间过大


影响范围：终端界面

期望结果：精简 UI，最大化终端内容显示区域



---

Bug #11：设置菜单 UI 缺陷（P3 - 低优先级）

问题描述：设置菜单 UI 未应用圆角，与整体设计风格不一致

影响范围：设置页面

期望结果：菜单样式符合统一设计规范



---

Bug #12：顶部操作菜单浮窗定位偏移（P2 - 中优先级）

问题描述：
SSH 终端顶部功能区（文件/监控/docker 等模块）的“更多”按钮（三个点图标）点击后，弹出的浮窗未以按钮为中心进行水平居中对齐，存在视觉偏移。

影响范围：
所有顶部带“三个点”操作按钮的模块弹出菜单

期望结果：
浮窗的水平中心轴应与触发按钮的水平中心轴完全重合，实现视觉上的垂直居中对齐。


---
优先级说明

P0：核心功能不可用 / 直接影响产品可用性（必须优先修）

P1：重要功能异常 / 明显影响体验或安全

P2：体验优化类问题

P3：视觉/UI细节问题
